# Autonomous Business Agent 🚀

This is an AI-powered agent built with **LangGraph** and **Cohere** that autonomously plans and executes business goals.

## How to Run Locally
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `COHERE_API_KEY` to a `.env` file.
4. Run: `streamlit run ui.py`
